## Início Rápido

1. **Obtenha seu token de API** no dashboard após aprovação da sua solicitação
2. **Configure a autenticação** usando Bearer token no header `Authorization`
3. **Crie seu primeiro pagamento** usando o endpoint `/payment/create`
4. **Verifique o status** com o endpoint `/payment/verify/{correlationID}`

<Note>
  Todas as requisições devem incluir o header `Authorization: Bearer {seu_token_api}` e usar a base URL `https://flowapplications.com.br/api/v1`
</Note>
